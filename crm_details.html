CRM System

A desktop application for managing customer relationships and sales pipelines.

About This Project

This project is a deep dive into data management. It focuses on creating an intuitive interface for tracking leads, managing communication logs, and visualizing the sales pipeline. It's built with performance and the user in mind.

Key Features

  • Contact Dashboard
  • Lead and Deal Tracking
  • Communication Log
  • Task & Follow-up Reminders
  • Built with Python and Tkinter

Technical Challenges

The main challenge was designing a database schema that was both robust and flexible. Implementing a search and filter system that felt instant was a key learning experience in optimizing user experience.
